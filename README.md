# GlobalEdge Remote Teams — Landing Page

A pixel-matched recreation of the GlobalEdge landing page, built with:

- **Vite** — dev server & build tool
- **Tailwind CSS** — utility-first styling
- **Alpine.js** — mobile menu interactivity

## Getting started

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

## Notes

- Photos use placeholder images (`pravatar.cc` / `picsum.photos`) since the
  original site's real photography wasn't available — swap the `src` values
  in `index.html` for your real assets.
- Colors, type scale, and layout are defined in `tailwind.config.js` and
  `src/style.css`.
